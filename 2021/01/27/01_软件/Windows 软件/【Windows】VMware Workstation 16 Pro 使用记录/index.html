Linux 安装 vmtools

  1. 装载VMware Tools 虚拟磁盘：虚拟机 > 安装（或升级）VMware Tools
  2. 挂载VMware Tools 虚拟 CD-ROM 映像（需创建装载点：mkdir /mnt/cdrom）:mount /dev/cdrom /mnt/cdrom
  3. 转到工作目录：cd /tmp
  4. 解压缩安装程序：tar zxpf /mnt/cdrom/VMwareTools-x.x.x-yyyy.tar.gz
  5. 以root身份安装VMware Tools：
  1. 卸载 CD-ROM 映像：umount /dev/cdrom

安装过程中遇到一些问题：

  1. 报错内容：bash: ./vmware-install.pl: /usr/bin/perl: bad interpreter: ......

    解决方案：yum groupinstall "Perl Support"

  2. 报错内容：Searching for GCC... The path "" is not valid path to the gcc binary.

    解决方案：yum -y install gcc

  3. 报错内容：Searching for GCC... The path "" is not valid path to the gcc binary.

    解决方案：yum -y install kernel-headers kernel-devel

共享文件夹

启用共享文件夹

  1. 选择该虚拟机，然后选择虚拟机 > 设置。
  2. 在选项选项卡中，选择共享文件夹，文件夹共享 选择 总是启用，添加 文件夹 并保存即可。

Linux 挂载共享文件夹

4.0 之前的 Linux 内核 4.0 及更高版本的 Linux 内核 说明
mount -t vmhgfs .host:/ /home/user1/shares /usr/bin/vmhgfs-fuse .host:/ /home/user1/shares -o subtype=vmhgfs-fuse,allow_other 将所有共享装载到 /home/user1/shares
mount -t vmhgfs .host:/foo /tmp/foo /usr/bin/vmhgfs-fuse .host:/foo /tmp/foo -o subtype=vmhgfs-fuse,allow_other 将名为 foo 的共享装载到 /tmp/foo
mount -t vmhgfs .host:/foo/bar /var/lib/bar /usr/bin/vmhgfs-fuse .host:/foo/bar /var/lib/bar -o subtype=vmhgfs-fuse,allow_other 将共享 foo 中的子目录 bar 装载到 /var/lib/bar

Linux 移除共享文件夹

Windows下VMware虚拟机开机自启

  1. 创建自启动脚本文件startup.bat
  1. 打开组策略（运行->执行gpedit.msc)，选择计算机配置->Windows配置-脚本(启动/关机),修改启动属性，添加脚本；

Windows下VMware虚拟机关机自动关闭

  1. 虚拟机完成 vmware tools 安装
  2. 创建自动关闭脚本shutdown.bat
  1. 打开组策略（运行->执行gpedit.msc)，选择计算机配置->Windows配置-脚本(启动/关机),修改关机属性，添加脚本；
