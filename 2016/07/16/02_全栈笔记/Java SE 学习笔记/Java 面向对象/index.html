面向对象的思想

类(Class)和对象(Object)是面向对象的核心概念。

  • 类是对一类事物的描述，是抽象的、概念上的定义
  • 对象是实际存在的该类事物的每个个体，因而也称为实例(instance)。

问题：面向对象与面向过程？

  • 二者都是一种思想，面向对象是相对于面向过程而言的。面向过程， 强调的
    是功能行为，以函数为最小单位，考虑怎么做。 面向对象，将功能封装进对
    象， 强调具备了功能的对象，以类/对象为最小单位，考虑谁来做。
  • 面向对象更加强调运用人类在日常的思维逻辑中采用的思想方法与原则，如
    抽象、分类、继承、聚合、多态等。

  • public：访问权限最宽，可以跨包访问，意为“公开的”。
  • protected
    • 父类的protected成员是包内可见的，并且对子类可见。
    • 若子类与父类不在同一个包中，那么在子类实例中，只能访问子类继承的父类的protected成员，而不能访问父类实例的protected成员。
  • private：被修饰的类、属性、方法只能由其对象访问。
  • default ： 被修饰的类、属性、方法只能在同一个包中访问。
